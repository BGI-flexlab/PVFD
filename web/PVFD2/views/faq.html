FAQ

Question 1: What Service can I get?

A : This website provides variant frequency querying and statistics service.Users can search on DATA page,and check out data statistics on Statistics page.

Question 2: How to query?

A:On home page or DATA page,you can input a chromosome position or dbSNP ID to search variation frequency information ,including various dimension distribution on ethnicity,area,gender and country.
